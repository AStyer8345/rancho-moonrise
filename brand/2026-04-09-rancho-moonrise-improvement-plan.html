SEO/AEOSearch & AI Visibility — Ongoing

Starts immediately · Ongoing weekly · Accelerates after DNS cutover

Rancho Moonrise does NOT rank for any non-brand search query. "Best glamping near Austin," "wedding venues Austin TX ranch," "pool day pass Austin" — zero presence. AI answer engines (ChatGPT, Perplexity, Google AI Overviews) don't cite Rancho Moonrise at all. The Vercel site has good schema and page structure — but it's not live yet. Launching the new site on the main domain is the single biggest SEO/AEO unlock.

S1Split blog posts into individual pages with unique URLs
ClaudeSEO: F → C

All 6 blog posts live on a single blog.html page — Google can't index them individually. Each post needs its own HTML file in site/blog/ with Article schema, canonical URL, meta description, and links back to the blog index. Blog index gets CollectionPage schema. This is a prerequisite for any blog SEO.

S2Add answer-first H2 summaries to key pages
ClaudeAEO

AI engines extract the first sentence after each H2. Currently, page content buries the answer. Add a direct, extractable answer in the first sentence of every H2 section on: glamping page, weddings page, corporate retreat page, accommodations page. Example: "Rancho Moonrise is a 34-acre glamping ranch 20 minutes from downtown Austin with luxury safari tents, cabins, a pool, and event spaces for up to 200 guests."

S3Build topical authority clusters with internal linking
ClaudeSEO + AEO

Pages exist but don't interlink deeply. Build three clusters: (1) Glamping cluster: glamping page ↔ packing guide ↔ glamping vs camping ↔ Austin outdoor activities, (2) Wedding cluster: weddings ↔ ranch wedding tips ↔ wedding venues near Austin ↔ bachelorette, (3) Events cluster: events ↔ corporate retreats ↔ host your event ↔ things to do Manor TX. Each page should link to 3-5 related pages with contextual anchor text.

S4Add BreadcrumbList + SpeakableSpecification schema
ClaudeAEO

Add BreadcrumbList schema to all subpages (Home > Glamping > ...). Add SpeakableSpecification to homepage, glamping, weddings, and corporate pages — this tells AI engines exactly which text blocks are quotable answers. Also add AggregateRating schema pulling from the 125 Google reviews (4.9★).

S5Create "Pool Day Pass Austin" dedicated page
ClaudeSEO + Revenue

ResortPass already ranks for "pool day pass Austin" with a Rancho Moonrise listing. Own this query directly with a dedicated page on the new site — pricing, hours (10AM-6PM), what's included, photos, link to ResortPass booking. This is high-margin, low-effort revenue with search demand.

S6Post-launch: Google Search Console + sitemap + indexing
AdamClaudeSEO: F → B

After DNS cutover: (1) Set up Google Search Console for ranchomoonrise.com, (2) Submit XML sitemap, (3) Request indexing of all key pages, (4) Set up redirect map from old BofillTech URLs, (5) Monitor indexing weekly — all pages should be indexed within 14 days. This is the moment everything built above starts working.

S7AEO baseline + monthly tracking
ClaudeAUTOAEO

Baseline (April 9, 2026): Rancho Moonrise is NOT cited in any AI answer engine for non-brand queries. Monthly task: query ChatGPT, Perplexity, and Google AI Overviews for all 10 target keywords and log whether Rancho Moonrise appears. Track competitors who DO appear. Target: cited in at least 3 of 10 queries within 90 days of site launch.
